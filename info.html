Tik Tak Toe

🎮 Spielablauf

  • Das Spiel wird auf einem 3×3-Feld gespielt.
  • Zwei Spieler treten gegeneinander an.
  • Die Spieler setzen abwechselnd ein X oder ein O in ein freies Feld.

🏆 Ziel des Spiels

  1. Drei eigene Zeichen in eine Reihe bringen.
  2. Die Reihe kann waagerecht →, senkrecht ↓ oder diagonal ↘ sein.
  3. Wer zuerst drei Zeichen in einer Reihe hat, gewinnt.

🧠 Strategien

  • Die Mitte ist oft ein guter Startpunkt.
  • Versuche, mehrere Gewinnmöglichkeiten gleichzeitig zu schaffen.
  • Blockiere rechtzeitig die Reihen deines Gegners.

⭐ Punkte verteilung (nur bei KI)

  • Sieg: +2 Punkte
  • Niederlage: 0 Punkte
  • Untentschieden: +1 Punkt
  • Die Punkte werden zurückgesetzt bei ...
    • - einer Niederlage
    - neustarten einer Runde
    - verlassen der Runde
